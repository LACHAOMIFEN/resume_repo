# RepoPilot Roadmap (Step-by-step)

## Phase 1 — Local MVP (Day 1-3)
1. Define role schemas (`agents.py`)
2. Implement linear workflow (`workflow.py`)
3. Add CLI (`main.py`) and mock execution
4. Add tests for workflow transitions

## Phase 2 — Engineering Gates (Day 4-6)
1. Add linter/test command execution wrappers
2. Fail-fast + retry policy
3. Structured logs (JSON)

## Phase 3 — GitHub Integration (Day 7-10)
1. `github.py`: fetch issue content by number/url
2. Generate branch/patch output
3. Create PR draft body markdown

## Phase 4 — Resume Polish (Day 11-14)
1. Demo script + screenshots
2. Metrics: success rate, avg runtime, merge-readiness
3. Final README and architecture diagram
